# Mini Diário PHP

## Resumo

Criar um CRUD mínimo de notas usando arquivo JSON.

## Tecnologias

PHP + HTML/CSS

## Como rodar

```bash
php -S localhost:8000 e acesse http://localhost:8000
```

## O que esse projeto mostra

- Formulário POST
- Persistência em JSON
- Escape de HTML com htmlspecialchars

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro formulário post e uma interface simples para usuário final.
