# Como rodar tudo

## Web: Paint, Sudoku e Corrida

Abra o `index.html` de cada pasta no navegador. Não precisa instalar dependências.

## Java - Mini Editor

```bash
cd 04-mini-editor-java
javac MiniEditor.java
java MiniEditor
```

## Python - Mini Tarefas

```bash
cd 05-mini-tarefas-python
python tarefas.py add "estudar javascript"
python tarefas.py list
python tarefas.py done 1
```

## Kotlin - Mini IMC

```bash
cd 06-mini-imc-kotlin
kotlinc Main.kt -include-runtime -d imc.jar
java -jar imc.jar
```

## Flutter - Mini Metas

```bash
cd 07-mini-metas-flutter
flutter create .
flutter run
```

## PHP - Mini Diário

```bash
cd 08-mini-diario-php
php -S localhost:8000
```

Depois acesse `http://localhost:8000` no navegador.
