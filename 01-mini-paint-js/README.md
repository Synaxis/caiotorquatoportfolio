# Mini Paint JS

## Resumo

Criar um quadro de desenho simples, responsivo e com exportação em PNG.

## Tecnologias

HTML, CSS e JavaScript puro

## Como rodar

```bash
Abra o arquivo index.html no navegador.
```

## O que esse projeto mostra

- Canvas API
- Pointer Events para mouse/caneta/toque
- Download do desenho em PNG

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro canvas api e uma interface simples para usuário final.
