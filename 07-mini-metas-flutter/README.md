# Mini Metas Flutter

## Resumo

Criar uma tela mobile curta para contar metas concluídas.

## Tecnologias

Flutter/Dart

## Como rodar

```bash
Dentro da pasta: flutter create . && flutter run
```

## O que esse projeto mostra

- StatefulWidget pequeno
- Material Design
- Controle de estado com setState

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro statefulwidget pequeno e uma interface simples para usuário final.
