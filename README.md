# Mini Portfolio - Caio Torquato

Pacote de mini projetos curtos, feitos para portfolio e estudo. A ideia é mostrar variedade: web, desktop, terminal, mobile e backend, mantendo o código principal com no máximo 30 linhas por arquivo.

## Projetos incluídos

1. **Mini Paint JS** - desenho em canvas com exportação PNG.
2. **Mini Sudoku 4x4** - lógica de jogo com validação no navegador.
3. **Mini Corrida** - joguinho canvas com colisão e pontuação.
4. **Mini Editor de Texto Java** - app desktop com abrir/salvar/limpar.
5. **Mini Tarefas Python** - lista de tarefas no terminal com JSON.
6. **Mini IMC Kotlin** - cálculo simples em console.
7. **Mini Metas Flutter** - contador mobile com Material Design.
8. **Mini Diário PHP** - CRUD mínimo de notas em arquivo JSON.
