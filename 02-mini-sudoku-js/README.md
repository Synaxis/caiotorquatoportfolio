# Mini Sudoku 4x4

## Resumo

Mostrar lógica de validação de jogo em uma versão 4x4 rápida de Sudoku.

## Tecnologias

HTML, CSS e JavaScript puro

## Como rodar

```bash
Abra o arquivo index.html no navegador.
```

## O que esse projeto mostra

- Manipulação de DOM
- Validação de entrada
- Feedback visual de erro/acerto

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro manipulação de dom e uma interface simples para usuário final.
