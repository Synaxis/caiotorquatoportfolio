# Mini IMC Kotlin

## Resumo

Criar um programa console limpo com cálculo e classificação de IMC.

## Tecnologias

Kotlin

## Como rodar

```bash
kotlinc Main.kt -include-runtime -d imc.jar && java -jar imc.jar
```

## O que esse projeto mostra

- Entrada segura com readlnOrNull
- Uso de when
- Funções pequenas e claras

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro entrada segura com readlnornull e uma interface simples para usuário final.
