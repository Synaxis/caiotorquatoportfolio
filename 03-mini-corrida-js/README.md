# Mini Corrida

## Resumo

Criar um jogo curto de desvio de obstáculos em canvas.

## Tecnologias

HTML, CSS e JavaScript puro

## Como rodar

```bash
Abra o arquivo index.html e use setas ou A/D.
```

## O que esse projeto mostra

- Game loop com requestAnimationFrame
- Detecção simples de colisão
- Controles por teclado

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro game loop com requestanimationframe e uma interface simples para usuário final.
