# Mini Editor de Texto

## Resumo

Mostrar uma aplicação desktop curta com abrir, salvar e limpar texto.

## Tecnologias

Java Swing

## Como rodar

```bash
No terminal: javac MiniEditor.java && java MiniEditor
```

## O que esse projeto mostra

- Interface desktop com Swing
- Leitura e escrita de arquivos
- JFileChooser e tratamento básico de erro

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro interface desktop com swing e uma interface simples para usuário final.
