# Mini Tarefas CLI

## Resumo

Criar uma lista de tarefas simples em terminal com persistência em JSON.

## Tecnologias

Python

## Como rodar

```bash
python tarefas.py add "estudar JS" | python tarefas.py list | python tarefas.py done 1
```

## O que esse projeto mostra

- CLI simples
- Persistência em JSON
- Comandos add/list/done/clear

## Explicando

> Eu fiz um projeto pequeno para treinar clareza de código. A regra foi manter o código principal curto, mas ainda entregar uma função real. Aqui eu mostro cli simples e uma interface simples para usuário final.
